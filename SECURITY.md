# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Grove, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

### How to Report

Use [GitHub Security Advisories](https://github.com/LeahArmstrong/grove-cli/security/advisories/new) to privately report the vulnerability.

### What to Expect

- **Acknowledgment**: Within 48 hours of your report
- **Triage**: Within 7 days, we will assess the severity and impact
- **Resolution**: We will work on a fix and coordinate disclosure

### Scope

This policy applies to the Grove CLI tool and its official plugins. Third-party plugins are outside the scope of this policy.

## Supported Versions

As Grove is pre-release software, only the latest version on the `main` branch is supported with security updates.
